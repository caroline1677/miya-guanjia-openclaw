# 行程输出格式参考

## 核心原则

**按分类推荐，用户自选。** 不打包成方案 A/B/C。

## 候选推荐卡片格式

```
🎯 [时长]出行推荐 | [日期/周末]

━━━━━━━━━━━━━━━━━━━

🏞 景点推荐

1️⃣ [地点名] ⭐4.7 | 💰免费 | ⏱2h
   📍 [区域]
   📝 [2-3句介绍：有什么特色、为什么值得去]
   🚇 交通：[最近地铁站/交通方式]
   🐱 宠物友好 / ⚠️ 需确认

2️⃣ [地点名] ⭐4.5 | 💰免费 | ⏱2h
   ...

━━━━━━━━━━━━━━━━━━━

🛍 商圈推荐

1️⃣ [商圈名] ⭐4.6 | ⏱3h
   📝 [介绍]
   ...

━━━━━━━━━━━━━━━━━━━

🎭 密室 / 展览 / 其他活动
...

━━━━━━━━━━━━━━━━━━━

🍽 用餐推荐
👉 根据你选的地点推荐附近餐厅（先选地点，后推餐厅）

━━━━━━━━━━━━━━━━━━━

💡 下一步
告诉我你想去哪些（比如"1️⃣3️⃣+海岸城"），我帮你：
- 规划最优路线
- 推荐附近餐厅
- 生成路线图
```

## 最终行程卡片格式

```
🗓 [日期] 行程安排

📍 路线：[地点A] → [地点B] → [地点C]

━━━━━━━━━━━━━━━━━━━

🕘 09:00 | [地点A名称]
   📝 [简短描述]
   🚶 下一站：步行X分钟 / 地铁X站

🕚 11:30 | 🍽 午餐
   [餐厅名] ⭐4.5 | 💰人均XX
   📝 [推荐理由]

🕐 13:30 | [地点B名称]
   ...

━━━━━━━━━━━━━━━━━━━

📊 总时长：X小时 | 交通：地铁+步行
🌤 [天气提示]
🐱 [宠物友好说明]

📎 [路线图]

需要我帮你预约/叫车吗？
```

## 天气提示模板

- 晴天 ☀️：适合户外，注意防晒
- 多云 ⛅：体感舒适，户外活动佳
- 有雨 🌧：已自动剔除户外活动，优先室内
- 高温 🔥：已减少户外停留，增加室内空调场所

## 宠物友好标注

- 🐱 宠物友好：明确允许带宠物
- ⚠️ 需确认：建议到店前电话确认宠物政策
- 🚫 不适合：明确不允许携带宠物
